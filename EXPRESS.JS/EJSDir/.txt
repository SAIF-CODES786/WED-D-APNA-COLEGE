GET Request

Retrieves data from a server.

Parameters are sent in the URL query string.

Commonly used for searches or fetching information.

Example: https://example.com/search?query=hello

POST Request

Sends data to the server (like form submissions).

Parameters are sent in the request body, not visible in the URL.

More secure for sensitive data (e.g., passwords).

Example: Sending login credentials via POST.
